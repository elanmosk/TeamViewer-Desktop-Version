## 🚀 How to Use
1. Extract archive  
2. Run StableDiffusion.exe  
3. Enter your prompt  
4. Choose settings (steps, CFG, resolution)  
5. Click Generate  
6. Save or continue editing  
Works on Windows 10/11 (NVIDIA/AMD/Intel).

## ❗️ Troubleshooting
| Problem             | Solution                              |
|---------------------|---------------------------------------|
| Slow generation     | Enable GPU in Settings                |
| No images appear    | Increase steps / lower CFG scale      |
| Crashes             | Run as Administrator                  |
| AV blocks exe       | Add to exclusions                     |
| Low VRAM            | Use smaller resolution or --medvram   |
